# Active Work

_The project's agent-agnostic re-entry point for work spanning multiple sessions — any agent (a teamlead engagement, a solo session, a project-specific agent) reads it to pick up the big work mid-stream. Rewritten as a fresh one-screen snapshot by whichever agent advances the work — never appended (git is the history). A trivial quick-fix may legitimately leave it untouched; that's expected, not a bug._

**Last shipped:** _What landed and was verified/merged this block, in prose — plus any new runtime gotcha (e.g. a new system dependency). One short paragraph._

## Goal (north star)

_The durable "why" — the outcome this block of work drives toward. Set when a batch is engaged, or by whichever agent picks up the work._

## Recommended sequence

_The ordered path to the goal. Strike through done steps in place; flag the next one. Cite roadmap rows / `BUG-NNN` ids rather than copying them._

## Next-session prompt

_Copy-paste resume: re-establish ground truth (branch, status, tests) → which roadmap rows → which deferred items / bugs to fold in → which skill/approach to open with → where to start._
